üöÄ AWS ECS

Elastic Container Service

Task Definition:

# Create task definition
aws ecs register-task-definition
--cli-input-json
file://task-def.json

# Create service
aws ecs create-service
--cluster my-cluster
--service-name my-service
--task-definition my-task
--desired-count 3

Features:

  • Serverless with Fargate
  • EC2 launch type
  • Service auto-scaling
  • Load balancer integration
  • CloudWatch monitoring

üìã Quick Reference Commands

Docker Basic Commands:

docker ps # List running containers
docker images # List images
docker pull nginx:latest # Pull image
docker run -d -p 8080:80 nginx # Run container
docker stop container_id # Stop container
docker rm container_id # Remove container
docker logs -f container_id # Follow logs
docker exec -it container_id bash # Access container shell

Docker Compose Commands:

docker-compose up -d # Start services
docker-compose down # Stop and remove
docker-compose ps # List services
docker-compose logs -f # Follow logs
docker-compose restart # Restart services
docker-compose build # Build images

Kubernetes Commands:

kubectl get pods # List pods
kubectl get services # List services
kubectl describe pod pod-name # Pod details
kubectl logs pod-name # View logs
kubectl apply -f deployment.yaml # Apply config
kubectl delete pod pod-name # Delete pod
kubectl exec -it pod-name -- bash # Access pod shell

Docker Swarm Commands:

docker swarm init # Initialize swarm
docker stack deploy -c stack.yml app# Deploy stack
docker service ls # List services
docker service scale app_web=5 # Scale service
docker stack rm app # Remove stack
docker node ls # List nodes
